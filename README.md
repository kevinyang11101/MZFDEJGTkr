# SpringBoot停车场管理系统 java892

## 项目概述

本项目是一款基于SpringBoot框架的停车场管理系统，实现前后端分离。该系统致力于通过高效的信息管理，提升停车场的运营效率。

## 技术栈

- 后端：SpringBoot + Mybatis + MySQL
- 前端：Vue

## 推荐环境配置

- 开发工具：IDEA
- JDK版本：1.8
- 项目管理：Maven
- 数据库：MySQL

## 功能模块

### 管理员端

- **用户信息管理**
  - 查询、新增、充值、修改、删除
  - 重置密码、分页
- **车位信息管理**
  - 查询、新增、修改、删除
  - 分页
- **车位费用设置**
  - 查询、修改
- **停泊车辆查询**
  - 查询、分页
- **车辆进出管理**
  - 查询、分页、驶出扣费
- **登录日志查询**
  - 查询、分页

### 用户端

- 登录、注册
- 首页
- **个人中心**
  - 信息修改
- **预定停车位**
  - 查询、预定、分页
- **缴费信息**
  - 信息查看、充值

## 系统角色

- 管理员
- 用户
